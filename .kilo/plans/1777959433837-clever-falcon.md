# Restore App Localization/Language Feature

## Summary
Re-implement translation support for restaurant configurations and contact section after accidental removal during refactoring.

## Files to Modify

### 1. Restaurant JSON Data Files
- `apps/web/restaurants/ramen-taro/data.json`
- `apps/web/restaurants/sushi-yama/data.json`

**Change:** Add `app` object at root level:
```json
"app": {
  "language": "JA",
  "currency": "JPY"
}
```

### 2. Create i18n Library
- `apps/web/lib/i18n.ts` (new file)

**Change:** Create `getTranslations` function that:
- Imports locale JSON files dynamically
- Returns translations object based on language code
- Provides fallback to English/default text

### 3. i18n Locale Files
- `apps/web/locales/en.json` (new file) - English base translations
- `apps/web/locales/ja.json` (new file) - Japanese translations

**Change:** Create translation files with contact section keys:
- `contact.title`, `contact.subtitle`, `contact.description`
- `contact.location`, `contact.phone`, `contact.email`, `contact.hours`
- Form labels: `contact.name`, `contact.email`, `contact.subject`
- Subject options: `contact.subject.reservation`, `contact.subject.event`, etc.
- Buttons: `contact.send`, `contact.processing`
- Alert messages: `contact.success`, `contact.error`

### 4. Contact Section Component
- `packages/ui/src/components/contact-section.tsx`

**Changes:**
- Add `translations?: Record<string, any>` to props interface
- Destructure `translations` in function parameters
- Replace all hardcoded English text with translation fallbacks:
  - SectionHeader props (subtitle, title, description)
  - Contact detail titles (Location, Phone, Email, Hours, Google Maps)
  - Form labels (Your Name, Email Address, Subject, Your Message)
  - Select dropdown options (Select a subject, Reservation Inquiry, Private Event, Feedback, Other)
  - Buttons (Request Reservation, Send Message, Processing...)
  - Alert messages in handleSubmit function
  - Social section header (Follow Us)
  - Service Hours section header
  - Holiday Notice text
  - Privacy Policy link text

### 5. Contact Page
- `apps/web/app/[restaurant]/contact/page.tsx`

**Changes:**
- Import `getTranslations` from `@/lib/i18n`
- Fetch translations: `const t = getTranslations(data.app?.language)`
- Pass translations to Navbar: `<Navbar translations={t} restaurant={...} />`
- Pass translations to ContactSection: `<ContactSection translations={t} ... />`
- Translate page header: `<h1>{t.contact?.title || "Contact Us"}</h1>`

### 6. Home Page
- `apps/web/app/[restaurant]/page.tsx`

**Changes:**
- Ensure translations are fetched: `const translations = getTranslations(data.app?.language)`
- Pass translations to ContactSection: `<ContactSection translations={translations} ... />`

### 7. Update RestaurantData Interface
- `apps/web/lib/restaurant.ts`

**Changes:**
- Add `app?: { language?: string; currency?: string }` to RestaurantData interface
