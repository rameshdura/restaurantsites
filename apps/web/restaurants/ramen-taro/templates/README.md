# Restaurant Asset Templates: Ramen Taro

This directory holds standardized assets for Ramen Taro to ensure consistent branding across browsers and social media.

## 1. Favicon (`templates/favicon/`)
- `favicon-32x32.png`: Main browser favicon.
- `favicon-16x16.png`: Smaller browser favicon.
- `apple-touch-icon-180x180.png`: For iOS home screen bookmarks.

## 2. Open Graph / Social Image (`templates/ogimage/`)
- `og-image-1200x630.png`: Standard size for Facebook, LinkedIn, and general social sharing.
- `twitter-summary-large-1200x600.png`: Optimized for Twitter/X cards.

## 3. App Icons (`templates/icon/`)
- `icon-192x192.png`: Android home screen icon.
- `icon-512x512.png`: Standard high-resolution icon.
